# Backend Laravel
- PHP 8.5.5
- Composer 2.5.7
- Laravel 10.10
- Autenticación: Sanctum (tokens)
- Base de datos: MySQL (Laragon)
- Testing: PHPUnit

# Reglas de código
- Código libre de comentarios.
- Siempre código factorizado.
- Utilizar herencia de ser posible.
- Preguntar antes de instalar paquetes.
- Estructurar carpetas para mejor organización si es necesario.

# Reglas de documentación
- Cada vez que se agregue o modifique una característica, actualizar README.md.
- El README debe documentar: característica implementada, librerías usadas, resumen de la lógica y decisiones de arquitectura.
- El README es la fuente de verdad del proyecto. Mantenerlo exhaustivo.
